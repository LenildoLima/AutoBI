# AutoBI

AutoBI is a multi-agent data analysis API capable of ingesting datasets,
detecting schema automatically, and generating KPIs, charts, tables and filters
dynamically.

## Features
- Automatic data ingestion
- Schema detection
- KPIs generation
- Charts suggestions
- Dynamic filters
- Multi-agent architecture

## Tech Stack
- Python
- FastAPI
- Pandas
- Uvicorn
